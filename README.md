## 🌟 Overview

**3D Quote Helper** is a high-performance 3D utility tailored for **CNC machining, sheet metal fabrication, and 3D printing services**. It eliminates the need for expensive CAD software by providing a fast, browser-based solution for **3D measurement**, **batch screenshots**, and **instant quotation data preparation**.

By leveraging **WebAssembly (Wasm)** and **WebGL**, the tool processes industrial files locally, ensuring extreme speed and 100% data privacy.

### 🚀 Key Features

* **Automated Archive Analysis**: Support for **ZIP and RAR** files. Simply drop a compressed package, and the tool automatically extracts and analyzes all 3D models within.
* **Batch 3D Measurement**: Automatically extract Max Bounding Box ($L \times W \times H$), **Surface Area**, and **Volume**—essential for 3D printing material estimation.
* **Fast 3D Screenshot**: Capture high-resolution images of models from standard angles for technical documentation and client quotes.
* **One-Click Excel Export**: Export all extracted dimensions and part metadata into a professional Excel sheet for instant quoting.
* **100% Privacy-First**: **Client-side processing.** Your CAD files are never uploaded to any server, keeping your sensitive IP secure.
* **Multi-Format Support**: High-precision parsing for industrial formats: `.stp`, `.step`, `.igs`, and 3D printing formats like `.stl`.
